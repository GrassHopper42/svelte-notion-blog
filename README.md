# Svelte + Notion API Blog

## Getting Started

make `.env` file in project root and input NOTION_API_KEY and NOTION_DATABASE_ID from notion integration.

```bash
NOTION_API_KEY=secret_********
NOTION_DATABASE_ID=*********
```

and install package

```bash
npm install
```

run

```bash
npm run dev
```
